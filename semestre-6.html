Disciplinas do 6° Semestre

Segurança da Informação (SEGA6) – 66,7h

Análise de aspectos de segurança de informação. Estudo de normas e políticas de segurança. Estudo de técnicas de segurança.

Segurança da Informação

Desenvolvimento para Web II (DW2A6) – 66,7h

Estudo e aplicação de linguagem de programação do lado servidor para desenvolvimento de sites dinâmicos com acesso a banco de dados.

Desenvolvimento para Web II

Projeto Integrado II (PI2A6) – 133,3h

Desenvolvimento de sistema com foco na análise do sistema. Análise da qualidade do projeto desenvolvido.

Projeto Integrado II

Tópicos Avançados (TPAA6) – 66,7h

Estudo de tópicos recentes em computação.

Tópicos Avançados
